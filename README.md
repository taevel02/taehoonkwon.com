# taehoonkwon.com

Personal archive and blog built with Remix and Vite.

## Tech Stack

- **Framework**: [Remix](https://remix.run/) (v2.17+)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Content**: Markdown based with custom preparation script
- **Icons**: [Remix Icon](https://remixicon.com/)
- **Typography**: [Pretendard](https://github.com/orioncactus/pretendard), Maru Buri (Local)
- **Deployment**: [Vercel](https://vercel.com/) (Analytics & Speed Insights integrated)

## Getting Started

### Prerequisites

- Node.js 22.x
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Features

- **PWA**: Ready via `vite-plugin-pwa`.
- **Performance**: Optimized for speed with local font hosting and local asset optimization.
- **Modern Styling**: Tailwind CSS v4 with custom theme support.
- **Dynamic Assets**: Automated RSS, Sitemap, and Metadata generation.
- **SEO**: Dynamic Open Graph image generation via Satori.
